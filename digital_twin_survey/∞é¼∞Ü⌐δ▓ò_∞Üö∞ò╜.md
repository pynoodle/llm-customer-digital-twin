# 📌 디지털 트윈 설문/인터뷰 시스템 - 빠른 사용법

## ⚡ 5분 시작 가이드

### 1단계: 설치 (3분)
```bash
# Mac/Linux
bash install.sh

# Windows
install.bat
```

### 2단계: API 키 설정 (1분)
`.env` 파일을 열어서 OpenAI API 키 입력:
```
OPENAI_API_KEY=sk-your-key-here
```

### 3단계: 데모 실행 (1분)
```bash
python demo.py
```

## 📊 사용 시나리오

### 시나리오 1: 빠른 테스트
```bash
python demo.py
```
- 3명에게 샘플 질문
- 결과 자동 저장

### 시나리오 2: 맞춤 설문
```bash
python digital_twin_survey_system.py
```
1. 응답자 선택 (필터링/샘플링)
2. 질문 입력
3. 실행 및 분석

### 시나리오 3: 고급 분석
```bash
python advanced_examples.py
```
- 특정 그룹 타겟팅
- 그룹 간 비교
- Excel 리포트

## 🎯 주요 명령어

### 기본 실행
```bash
python demo.py                        # 빠른 데모
python digital_twin_survey_system.py  # 전체 시스템
python advanced_examples.py           # 고급 예제
python analysis_tools.py              # 결과 분석
```

### 결과 확인
```bash
ls results/                          # 결과 파일 확인
cat demo_results_survey_1.csv        # CSV 내용 보기
```

## 🆘 문제 해결

### 데이터셋 로드 오류
```bash

```

### API 오류
```bash
# .env 파일에서 API 키 확인
cat .env
```

### 패키지 오류
```bash
pip install --upgrade -r requirements.txt
```
